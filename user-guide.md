# Руководство пользователя

Заходи на сайт: http://ip-адрес-сервера
Если сайт не работает - пиши админу
Для доступа к базе данных используй логин/пароль из файла configs/db-access.txt
Логи приложения лежат в /var/log/my-app/
Если нужно загрузить файлы - кидай в /var/www/uploads/
Бэкапы делаются автоматически каждый день в 2:00
При проблемах: перезагрузи страницу, почисти кэш браузера, проверь интернет
Если ничего не помогло - зови админа

Всё. Больше от пользователя ничего не требуется.
