# Описание файлов

## Скрипты
- `scripts/master-deploy.sh` - главный скрипт развертывания
- `scripts/vpn/` - установка и настройка VPN
- `scripts/monitoring/` - развертывание мониторинга  
- `scripts/backups/` - система бэкапов
- `scripts/security/` - настройка безопасности

## Конфиги
- `configs/vpn/` - конфигурации OpenVPN
- `configs/prometheus/` - настройки мониторинга
- `configs/grafana/` - дашборды Grafana

## Инфраструктура
- `infrastructure/servers.list` - инвентарь серверов
- `infrastructure/network/` - сетевые настройки
