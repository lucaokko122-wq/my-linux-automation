# Процесс бэкапов

## Что бэкапим
- Конфиги сервисов (/etc/)
- Данные приложений (/var/)
- Логи (/var/log/)
- Базы данных (дампы)

## Расписание
- **Ежедневно**: инкрементальные бэкапы
- **Еженедельно**: полные бэкапы  
- **Ежемесячно**: архивация в облако

## Восстановление
```bash
./scripts/backups/restore.sh [дата]
