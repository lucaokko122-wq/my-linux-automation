# Руководство пользователя VPN

## Подключение
1. Скачайте .ovpn файл с сервера
2. Импортируйте в OpenVPN клиент
3. Подключитесь используя логин/пароль

## Проверка подключения
- Статус: Active
- IP должен смениться на IP VPN сервера
- Доступ к внутренним ресурсам

## Устранение проблем
- Проверьте интернет соединение
- Убедитесь что файл конфига корректный
- Перезапустите OpenVPN клиент
